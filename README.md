# Calculatrice-Py

Calculatrice simple réalisée en Python avec la bibliothèque **Tkinter**.

## Fonctionnalités
- Opérations mathématiques de base (+, -, *, /).
- Interface graphique intuitive.
- Bouton "AC" pour effacer l'écran.

## Aperçu
![Aperçu de la calculatrice](Capture%20d’écran.png)

## Installation
Assurez-vous d'avoir Python installé sur votre machine.

```bash
python calcul.py
```
